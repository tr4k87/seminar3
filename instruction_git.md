## Подготовка репозитория

Чтобы создать (инициализировать) новый репозиторий в текущей папке нужно ввести в терминале команду:

    git init

## Создание коммитов

Для создания коммитов необходимо использовать комманду в терминале:

    git commit -m "Коммит"
